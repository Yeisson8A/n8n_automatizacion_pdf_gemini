# **Workflow automatizado en N8N**
Flujo para lectura de archivos PDF mediante una petición POST, extracción de información y conversión a formato JSON usando las herramientas N8N y Gemini AI.

<img width="1709" height="488" alt="Captura de pantalla 2025-12-01 135231" src="https://github.com/user-attachments/assets/7bbdd0df-db10-40c6-93b5-c6dc6ca9b7ac" />

## **Configuración**:

1. Para la creación del flujo se debe crear una cuenta en N8N Cloud: `https://app.n8n.cloud/login`

2. Para la conexión a Gemini AI, es necesario crear un proyecto y una API Key en Google AI Studio: `https://aistudio.google.com/`:
   - Opción Get API Key - Proyectos - Crear proyecto nuevo
   - Una vez creado el proyecto, dar clic en "Crear clave de API", seleccionar proyecto y por último "Crear clave"
